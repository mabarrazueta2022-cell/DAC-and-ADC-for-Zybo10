Este archivo contiene los códigos en vhdl de los componentes para controlar el ADC Pmod AD1 y el DAC Pmod DA2
Además contiene los bloques necesarios para realizar medidas sobre los datos recibidos por el ADC como el 
valor medio de la señal y la frecuencia.
Se agrega además el código en C para la generación de datos en el DAC y lectura de los datos del ADC y procesado
para la ejecución de medidas.
